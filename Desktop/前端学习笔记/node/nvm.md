#### nvm 是一款可以进行node管理版本的东西

###### 	1.安装(去github上下载)

###### 	2.命令

- version 版本

- list 可使用的版本
- current 当前选中的版本
- use 设置管理node的版本
- install 安装node
- uninstall 卸载
- nvm_node_mirror 节点镜像    使用*https://npmmirror.com/mirrors/node/* 
- nvm_npm_mirror npm镜像    使用*https://npmmirror.com/mirrors/npm/*
