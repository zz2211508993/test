发布项目

- 1.注册npm

- 2.在命令行登陆
  - npm login
- 3.修改package.json

- 4.发布到npm registry 上
  - ​	npm publish
- 5.更新仓库
  - 1.修改版本号 按照semver规范
  - 2.重新发布

删除发布的包

- npm unpublish

让发布的包过期

​	npm deprecate

package.json 配置 

- **keywords** 
- **homepage**  你的文章 没有就github中的
- **repository**
  - type 发布类型 gith等等
  - url 和homepage一样
- **name** 
- **version**
- **main**
